Props e componentes

Os componentes React são funções ou classes JavaScript que aceitam parâmetros, chamados props e retornar uma descrição da interface com base nesses parâmetros. Dessa forma, você pode renderizar os elementos HTML e os dados do aplicativo de uma só vez, sem escrever código adicional para atualizar valores dentro do HTML.

Os parametros das funções ou classes são as props

Os valores herdados são passados sendo que os props recebem os valores do state e são os props passados como parametros da função.Estes paramentros são as props como se as props fosse os parametros de uma função podendo ser utilizados ao suas funçóes serem chamadas.

Caso queira pegar as propriedades de uma propriedade e passar para os elementos filhos utilize o Children.
